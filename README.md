# ML Calories Burnt

ML Calories Burnt is a repository dedicated to estimating the number of calories burned during various activities using machine learning algorithms.

## Features
- Data preprocessing for metabolic and physical activity data
- Implemented machine learning models
- Insights and predictions on calories burned

## Get Started
1. Clone the repository:
   ```
   git clone https://github.com/priyanshupaikra/ml_calories_burnt.git
   ```
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the application or train the model based on dataset:
   ```
   python main.py
   ```
